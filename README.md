# create-rolldown

Scaffolding tool for Rolldown projects.

## Usage

```bash
# npm
npm create rolldown@latest

# pnpm
pnpm create rolldown

# yarn
yarn create rolldown

# bun
bun create rolldown
```

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build
npm run build

# Run tests
npm test
```
